# JBake

by **Jonathan Bullock**

JBake is a Java based static site/blog generator, bake your own web site!

## Getting Started

- Get the code and run mvn package to build the executable JAR.
- Run java -jar <jarfile> <source_folder> <destination_folder>

## Usage
